# Oswald

A POS System for small shops, available in kurdish and english.

### Stack

- Language: Typescript
- UI: React + Mantine + Heroicons
- State Manager: Redux Toolkit
- Translation: React-i18next
- Backend: Pocketbase

### Features

- Add, edit, and remove products with categories
